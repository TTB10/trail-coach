# Trail Coach

> AI-powered ultra-trail coaching platform: Strava sync, physiological load modeling (CTL/TSB/ACWR), GPX-based race prediction and adaptive pacing strategy.

## Stack

- **Backend:** FastAPI (Python 3.12) + PostgreSQL via Supabase
- **Frontend:** Next.js 15 (TypeScript) + Tailwind CSS + shadcn/ui
- **Auth:** Supabase Auth
- **Deployment:** Vercel (frontend) + Railway (backend) + Supabase (DB)
